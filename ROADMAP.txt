================================================================================
AGENT MANAGER GUI - DEVELOPMENT ROADMAP
================================================================================

PROJECT GOAL:
Build a fast, native Rust-based GUI tool for managing CLI agents (Gemini CLI,
Claude Code, etc.) with features for creating agents, subagents, and 
orchestrating multiple CLI tools simultaneously.

TECH STACK:
- GUI: egui + eframe (Pure Rust, immediate mode GUI)
- Backend: Rust (Tokio for async operations)
- Agent Framework: Custom build with Rusty Agent (for LLM integration)
- Process Management: tokio::process for CLI agent spawning

DEVELOPMENT WORKFLOW
================================================================================

1. Fast Iteration Setup:
   - Use `cargo watch -x 'run'` for auto-rebuild on save
   - Test GUI changes immediately
   - Iterate quickly on UI components

2. Testing Strategy:
   - Unit tests for agent management logic
   - Integration tests for CLI integration
   - Manual testing for GUI components

3. Code Organization:
   - src/
     - main.rs (GUI app entry point)
     - agent/ (agent management logic)
     - cli/ (CLI integration)
     - ui/ (UI components)
     - state/ (state management)
     - utils/ (utility functions)

4. Version Control:
   - Commit after each phase completion
   - Feature branches for major features
   - Keep main branch stable
